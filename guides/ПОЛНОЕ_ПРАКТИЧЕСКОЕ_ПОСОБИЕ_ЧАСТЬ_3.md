# 11. КЕЙСЫ, ОШИБКИ, СКРЫТЫЕ ТЕХНИКИ (продолжение)

## 11.3. Скрытые техники влияния (продолжение)

### Техника «Взаимный обмен»

Дайте что-то бесплатно, чтобы получить что-то взамен.

```
«Я подготовил для вас бесплатный предварительный анализ.
Это заняло у меня 2 часа.

Взамен я прошу только одного: если решите работать —
работать со мной. Договорились?»
```

### Техника «Нога в двери»

Начните с маленькой просьбы, переходите к большой.

```
Шаг 1: «Скиньте документы, я посмотрю» (легко)
Шаг 2: «Давайте созвонимся на 10 минут» (легко)
Шаг 3: «Оплатите первый этап — 5 000₽» (уже легче)
Шаг 4: «Оплатите второй этап — 15 000₽» (ещё легче)
```

### Техника «Ограниченный выбор»

Дайте клиенту выбор, где оба варианта выгодны вам.

```
«У вас есть два варианта:
1. Базовый пакет — 15 000₽, срок 2 недели.
2. Премиум пакет — 40 000₽, срок 5 дней.

Какой вам удобнее?»
```

### Техника «Потому что»

Всегда объясняйте, почему клиент должен что-то сделать.

```
«Оплатите этап, потому что без этого мы не можем начать.»
«Подпишите договор, потому что это защитит ваши интересы.»
«Скиньте документы, потому что без них нет доказательной базы.»
```

### Техника «Именные бонусы»

Дайте клиенту почувствовать себя особенным.

```
«[Имя], я сделаю для вас личный бонус:
бесплатную консультацию по налоговой оптимизации после возврата.

Это только для вас, потому что я вижу ваш потенциал.»
```

### Техника «Тройное подтверждение»

Заставьте клиента трижды подтвердить своё намерение.

```
1. «Вы действительно хотите вернуть деньги?» — ДА
2. «Вы готовы пройти все этапы?» — ДА
3. «Вы готовы начать прямо сейчас?» — ДА
```

### Техника «Тишина»

После того как назвали цену — **замолчите**.

```
Вы: «Стоимость этапа — 15 000₽»
     (молчите и ждёте)

Клиент: «...»
     (он должен ответить первым)

Кто первый заговорил после цены — тот проиграл.
```

### Техника «Реверсивная психология»

Скажите клиенту, что ему, возможно, не стоит работать с вами.

```
«[Имя], я должен быть с вами честен.
Наша система подходит не всем.

Если вы не готовы проходить этапы последовательно —
возможно, вам стоит поискать другие варианты.

Но если вы готовы довериться системе — результат будет.»
```

### Техника «Список преимуществ»

Перечислите все, что клиент получит.

```
«Что вы получаете, оплатив первый этап:

1. Полный анализ вашего дела
2. Заключение о шансах на возврат
3. Стратегию действий
4. Личного менеджера
5. Юридическую поддержку

Всё это за [сумма]. Это меньше, чем вы тратите на кофе в месяц.»
```

# ПРИЛОЖЕНИЕ: ШПАРГАЛКА МЕНЕДЖЕРА

## 10 золотых правил

```
1. Сначала слушай, потом говори
2. Сначала ценность, потом цена
3. Сначала доверие, потом оплата
4. Не обещай 100% — дай надежду
5. Дроби платежи — не проси всё сразу
6. Используй «потому что» — объясняй каждый шаг
7. После цены — молчи. Кто первый заговорил — тот проиграл
8. Давай выбор, где оба варианта выгодны тебе
9. Создавай срочность, но не дави
10. Всегда закрывай на следующий шаг
```

## 5 фраз, которые продают

```
1. «Это стандартная процедура для всех клиентов»
2. «Без этого этапа мы не можем продолжить»
3. «Это [процент]% от того, что вы потеряли»
4. «Если ничего не делать — вы потеряете всё»
5. «Решение за вами, но время работает против нас»
```

## 5 фраз, которые убивают продажу

```
1. «Я гарантирую 100% возврат»
2. «Платите и получите результат»
3. «Это очень сложно, вы не поймёте»
4. «Почему вы не пришли раньше?»
5. «Все так делают»
```

## Структура идеального звонка

```
1. Приветствие + представление (10 сек)
2. Вопрос-крючок (10 сек)
3. Слушаем историю клиента (2-3 мин)
4. Квалификация (2 мин)
5. Презентация системы (2 мин)
6. Работа с возражениями (3-5 мин)
7. Дожим на оплату (2 мин)
8. Закрытие на следующий шаг (1 мин)
```

# ЗАКЛЮЧЕНИЕ

## Что должен уметь менеджер после прочтения:

1. **Вести клиента с 0 до оплаты** — от первого звонка до получения денег
2. **Понимать, где зарабатывать** — на каждом этапе, на апсейлах, на LTV
3. **Уметь дожимать** — техники, скрипты, психология
4. **Контролировать процесс** — CRM, статусы, триггеры, уведомления

## Главная мысль:

> **Клиент, потерявший деньги, — это не жертва. Это человек, который готов заплатить, чтобы вернуть своё. Ваша задача — провести его по системе, где каждый шаг логичен, каждая оплата оправдана, а результат — возврат денег.**

## Помните:

- Вы не продаёте — вы помогаете вернуть
- Вы не давите — вы объясняете
- Вы не обещаете — вы даёте надежду
- Вы не берёте — вы зарабатываете
