# AntiFrog | Возврат активов из мошеннических платформ

Международный портал по борьбе с финансовым мошенничеством в Казахстане.

## Структура сайта

| Страница | Описание |
|----------|----------|
| `/` | Главная — статистика, услуги, реестр, квиз, кейсы, команда, контакты |
| `/registry.html` | Реестр компаний — анимированный поиск по 100+ мошенникам |
| `/cases.html` | Кейсы — 847 успешных возвратов |
| `/admin.html` | Админ-панель — управление данными через GitHub API |
| `/client/login.html` | Вход в кабинет клиента — с анимацией «поиск по реестрам» |
| `/client/dashboard.html` | Дашборд клиента — статус, прогресс, финансы, комментарии |
| `/schemes/` | Схемы обмана — 7 стадий + доп. схемы |
| `/documents/` | Фейковые документы — как распознать подделки |
| `/organizations/` | Фейковые организации — брокеры, регуляторы, юристы |
| `/guides/` | Полные пособия — 7+ документов для юристов и клиентов |

## Технологии

- **HTML/CSS/JS** — статический сайт для GitHub Pages
- **Tailwind CSS** — утилитарный CSS-фреймворк
- **GSAP** — анимации скролла, появления элементов, прогресс-бары
- **data.json** — единый файл данных (реестр, клиенты, кейсы, команда)

## Управление данными

1. Откройте `/admin.html`
2. Введите GitHub token (Personal Access Token)
3. Редактируйте data.json в интерфейсе
4. Нажмите «Push в GitHub»
5. Сайт обновится автоматически через 1-2 минуты
