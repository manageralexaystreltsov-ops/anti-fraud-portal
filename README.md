# ⚠ Анти-Фрод Казахстан

Полный образовательный портал по защите от мошенников.

## Структура

- `/` — главная: 7 стадий обмана + горячие линии
- `/schemes/` — полная карта схем обмана (11 схем)
- `/documents/` — фейковые документы: как распознать (5 типов)
- `/organizations/` — фейковые организации: как отличить (5 типов)
- `/guides/` — полные пособия (7 документов)

## Назначение

Образовательный контент для граждан Казахстана.
Покажите близким — знание защищает от мошенников.
