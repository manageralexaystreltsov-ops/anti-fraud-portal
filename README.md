# AntiFrog | Возврат активов

## Страницы

| Страница | Назначение |
|----------|-----------|
| `/` | Главная — статистика, шаги, услуги, CTA |
| `/apply.html` | Заявка — пошаговая форма (ФИО, ИИН, телефон, описание) |
| `/check.html` | Проверка дела — ввод номера/ФИО/ИНН → статус |
| `/services.html` | Услуги — полный перечень с ценами |
| `/admin.html` | Админ-панель — управление заявками (через URL) |

## Как работает

1. Клиент заполняет форму на `apply.html`
2. Заявка автоматически уходит в Telegram бот
3. Администратор видит заявку в `admin.html`
4. Клиент проверяет статус на `check.html` по номеру дела

## Технологии

- HTML/CSS/JS — статический сайт
- Tailwind CSS
- GSAP — анимации
- Telegram Bot API — доставка заявок
- localStorage — хранение данных (в браузере)
